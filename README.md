# Todo-List

### Usage
On your terminal

$ cd server/

$ virtualenv venv

$ source venv/bin/activate

$ pip install -r requirements.txt

$ Run python app.py

Open another tab on your terminal

$ cd static/

$ yarn start

Enjoy!!

If you have enough enjoyed, run the below command

$ deactivate
